## States

### Active (Current Page)
- Color: Primary-800 (#172d65)
- Icon Stroke: 2.5px
- Indicator: Orange dot (•)
- Top Bar: Positioned under active item
- Font Weight: Medium

### Inactive
- Color: Gray-500
- Icon Stroke: 2px
- Hover: Primary-600
- Font Weight: Normal

### Hidden (Scrolling Down)
- Transform: translateY(100%)
- Trigger: Scroll down >100px
- Restore: Scroll up
